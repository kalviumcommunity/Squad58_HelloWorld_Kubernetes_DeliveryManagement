# Frontend

React + Vite dashboard for delivery operations.

## Run

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Notes

- Uses `/api` proxy in local dev (configured in `vite.config.js`).
- Production container serves static assets with Nginx.
